# Mohammed ElSerry | Video Editor Portfolio

A professional portfolio website for **Mohammed ElSerry**, a video editor with 3+ years of experience specializing in cinematic storytelling, color grading, and creative post-production.

## Features

- **Organized Portfolio Categories**: 12 distinct categories showcasing different types of video editing work
- **Google Drive Integration**: Direct link to full portfolio hosted on Google Drive
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices
- **Modern UI/UX**: Sleek dark mode aesthetic with blue accent theme (#2c8aff)
- **Interactive Animations**: Powered by AOS (Animate On Scroll) for engaging visual effects
- **Smooth Navigation**: Fixed navigation bar with glassmorphism effect
- **Direct Contact**: Functional contact form with email, phone, and social media links (WhatsApp, Instagram)

## Technologies Used

- **Core**: HTML5, CSS3, JavaScript (Vanilla)
- **Libraries**: 
  - [AOS](https://michalsnik.github.io/aos/) - Animate On Scroll
  - [SwiperJS](https://swiperjs.com/) - Touch Slider
- **Icons**: [Font Awesome](https://fontawesome.com/)
- **Typography**: [Google Fonts](https://fonts.google.com/) (Inter, Syne, Space Grotesk)
- **Color Scheme**: Dark theme with blue accent (#2c8aff)

## Skills Highlighted

- Video Editing (Adobe Premiere Pro)
